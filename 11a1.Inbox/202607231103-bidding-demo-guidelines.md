Demo creation guidelines for bidding demos:

- Focus on what the **business needs**, not the code
- Keep it as simple as possible
- Static frontend is fine — minimal framework wrangling
- Use mock data as much as possible
- Backend logic should be simple; only build BE for the crucial interactive sections that demonstrate core capabilities
