# Hybrid Interview Answers

Strong interview answers often combine multiple frames so one answer shows impact, reasoning, execution, and learning.

A flexible hybrid:

1. Impact — state the result.
2. Problem — explain why it mattered.
3. Decision — name the options and tradeoff.
4. Story — briefly describe implementation.
5. Reflection — state the lesson learned.

Use a hybrid when a single frame would hide an important signal. For example, a senior project answer may need impact-first opening, decision-based middle, and story-based evidence.
