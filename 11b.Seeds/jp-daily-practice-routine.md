# Japanese daily practice — workshop into a real routine

## Situation
Learning Japanese across all skill dimensions (reading, writing, listening, speaking, grammar, vocab). The list below is a "floor" of activities — everything that *could* be done in a day — but it's a grab-bag, not a routine. Without tiers, bad days hit and the whole thing collapses. The need is to workshop this into a sustainable habit: the right size for available time and energy, wired as an atomic habit (trigger → action → reward), and backed by accountability so it doesn't drift.

## Key insight
An all-or-nothing daily checklist is self-defeating. The highest-leverage design decision is the **worst-day minimum**: 3–5 non-negotiable activities that cover the core skill dimensions and take ≤15 minutes. Everything else is optional extension. Pair that with an **atomic habit loop** (trigger → routine → reward) and an **accountability hook**, and the minimum becomes self-propelling rather than willpower-dependent.
